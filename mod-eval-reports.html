CLEAR Learning Evaluation Dashboard

Upload, filter, analyze, and export module evaluation data

Getting Started

  1. Navigate to one or both Excel files below. Each file is hyperlinked to it's location in Amanda Vance's OneDrive. The tool reads the Sheet1 tab from each file.
    • ILT file contains instructor-led training evaluations (includes instructor ratings).
    • Async file contains self-study/asynchronous module evaluations.
  2. Create a local copy of the files.
  3. Drag and drop the local files to the appropriate drop zones below.
  4. Once loaded, use the Analyze tab for data exploration or the Instructor Reports tab for PDF/XLSX output.

NOTE: Asynchronous module evaluations from programs completed prior to this version of the Evaluation Dashboard (2026 02 & 2026 03 Cohorts) are located in the ILT file.

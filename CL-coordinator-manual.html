Program Coordinator Manual

Everything you need to manage CLEAR Learning program cohorts from event creation through close-out — for both in-person and virtual delivery.

This manual is organized by program, grouped into three categories: Board Member Training, Regulatory Administration, and Investigations & Inspections. Each program has its own page walking through its full lifecycle — from event creation through close-out — for however it's actually delivered (in-person, virtual, or both). Where a program has its own facilitation guide or exam FAQ, it's included right on that program's page.

The Universal Session Facilitation Guidelines apply across every program regardless of delivery mode. The Tools & Platforms section is a reference library of process guides, each opening in a new tab.

Phase 2
Set Up the Cohort
▼
  • Create the cohort folder in SharePoint
  • 📄 Save the Program and Instructor Agendas for the cohort to the folder — use the 🔧 Program Agenda Generator
  • Copy the PowerPoint files for the program from the CLEAR Learning current in-person files
  • 📄 Generate the QR codes for the module evaluations – save in the cohort folder in a "QR codes" subfolder, then update the QR codes in the PowerPoint slide decks for each module (update speaker notes as necessary)
  • 📄 Set up the live sessions for the cohort in TalentLMS — one live session per day of the program
  • 📄 Update the TalentLMS Welcome Email template with cohort-specific information — do this before importing the cohort
  • Update Slido for the cohort's live sessions
    • Reset all polls so every poll shows zero responses
    • Confirm the poll active-date window encompasses the full program dates — it defaults to Jan 1–Dec 31 of the current calendar year, so this only needs attention for the first program of a new calendar year
  • Send the slide decks to the instructors so they can review before class begins
  • Deactivate the final exam in the course before importing the cohort — the exam lives inside the main course, so if it's left active, participants can access it at any time. It gets reactivated on-site once exam instructions have been given (see Phase 5).
✦
Phase 3
Import the Cohort into TalentLMS
▼
  • 📄 Generate and upload the import file into TalentLMS using the 🔧 User Import Tool and watch for progress, done one week before the program. The upload file:
    • Creates the cohort
    • Assigns the courses to the cohort
    • Creates/updates user accounts
    • Assigns the cohort and the courses to the users — this also gives them access to the standard release and the course itself
  • Check that the welcome email from TalentLMS was sent — it tells participants to complete the standard release
  • Once upload is complete, 📄 register all users for all instructor-led sessions assigned to their cohort
✦
Phase 4
Pre-Program Reminders
▼
  • The Thursday before the program, send 📄 standard release reminders to anyone who hasn't completed it — use the 🔧 NCIT Standard Release Status Report tool

Why this matters for in-person only: the standard release blocks access to the course itself. Virtual participants notice immediately because they have to log into the portal to get the course content. In-person participants are sitting in a room and may never have logged into the portal — they won't discover they need to complete the standard release until they try to take the exam. Sending this reminder in advance avoids 50 people trying to complete the standard release at check-in.

On-Site Cohort Management
▼
  • Check in with venue
  • Set up venue meeting space
  • Check in attendees
  • Give introductory remarks at beginning of program

  • Provide customer service and technical support
  • Assist instructors as needed
  • Provide exam instructions and answer any questions
  • Once all exam instructions have been covered and the review session is complete, reactivate the final exam (deactivated back in Phase 2)
  • Proctor the exam in person
  • Pack up CLEAR property and prepare for shipping back to HQ

Close Out the Cohort
▼
  • 📄 Pull the NCIT Exam status report using the Exam Report Generator — use the 🔧 Exam Report Generator tool

    • Import payment verification from Novi for the cohort
    • Export the Credly upload template and import to issue badges
  • Communicate with attendees who fail the exam and follow retesting procedure
  • Verify certificate eligibility and process requests using the 📄 NCIT Supervisor Letter Dashboard — once eligibility is verified, mark the course complete per that process document
  • Pull module evaluation reports and share with instructors — use the 🔧 Module Evaluation Report Dashboard (see 📄 process document)
  • 📄 Mark attendance in Novi once attendees have completed all course compenents and passed the final exam. This action will be used to determine NCIT Specialized eligibility.
  • Instructor emails — send instructors their evaluation report for the cohort

NCIT Basic Program
NCIT Basic — All Sessions
▼

The NCIT Basic Program consists of a Welcome Session, five instructor-led content sessions, two Q&A sessions, a Program Review, and an exam delivered over two weeks. All times are in Eastern Time (ET).

Welcome Session
Purpose & Key Topics Orients new participants to program expectations, community guidelines, and the learning portal. No instructor — facilitated entirely by CLEAR Learning staff. Cover: CLEAR community guidelines and disclaimer, program agenda and session schedule, learning portal walkthrough (learner info, calendar view, participant guides, self-study, knowledge checks, module/program evaluations, instructor-led sessions, Module 5 Discussion Board, application for certification), standard release form reminder, and contact info.
Facilitator Notes
  • Session is recorded for CLEAR staff training purposes only — inform participants at the start.
  • Post program agenda in the chat.
  • Leave ample time for Q&A — participants often have portal navigation questions.
  • Encourage email over phone for support (staff may be in sessions).
Session 1 — Investigative Process
Pre-Session Notes
  • First live instructor-led session — participants receive the participant guide here.
  • One optional poll (Slide 19 — Investigative Process).
Opening Script "Good afternoon, everyone! Welcome to the first Instructor Led session of the NCIT Basic Program! My name is [Facilitator Name], and I'll be your session facilitator for the Investigative Process session."

NCIT Exam — Frequently Asked Questions
▼

Use these answers during Q&A sessions or when responding to participant emails about the exam.

What is the exam format?▼
The exam is pass/fail — no direct score is given to participants.
Can participants schedule for the same day?▼
No. Prometric does not allow same-day scheduling — the earliest test date is the next day.
When does the testing window open and close?▼
The testing window is 2 weeks, beginning the day of the Program Review Session. Advise participants not to schedule on the last day of the window.
Can a participant test in another state?▼
Yes, if their agency allows it.
How can participants check their system compatibility?▼
Direct them to https://rpcandidate.prometric.com/ — this tests their device only. They should also confirm their agency firewall/IT separately.
What ID is accepted?▼
Driver's license or passport is recommended. Large jewelry must be removed per Prometric security policy. No laptop on a docking station.
When will participants receive their digital badge?▼
Issued within a few business days after the testing window closes — no later than 3 weeks from the Review Session.
What is required for the NCIT Basic certificate?▼
Passing the exam plus 1 year of regulatory investigation/inspection experience. Records do not expire if the participant waits to apply.
How does the Specialized badge/certificate work?▼
Pass 1–2 sections → individual badges for those sections. Pass all 3 → one badge listing all modules. No expiration on NCIT Basic or Specialized certificate.
What is the retake policy for NCIT Basic?▼
If a participant fails, they may retake through Prometric after paying the $100 rescheduling fee. They do not need to retake the full course.
What is the retake policy for NCIT Specialized?▼
  • Fail 1–2 sections → retake internally with a CLEAR staff member ($50 for 1 section, $75 for 2).
  • Fail all 3 → retake through Prometric ($100 fee).
When does learning portal access close?▼
Access closes at the end of the testing window. Participants should download the eBook before then. Supplemental course recordings remain available until the course expires.
Will participants be tested on anything not in the eBook?▼
No. Participants will not be tested on anything not covered in the eBook/participant guide. Knowledge Assessments are locked and allowed only once — similar to exam questions. No practice tests are available. The Review Session is not recorded.
What if the name on the certificate is wrong?▼
The badge matches the name provided for the exam. Participants should contact CLEAR after receipt to make changes. Note: Prometric is instructed not to check middle names; if the middle name appears on the same line as the first name, the participant should email clearlearning@clearhq.org.
What if a module shows incomplete in the portal?▼
Contact CLEAR Learning — staff can mark it complete.
Can participants get an extension?▼
Reach out to CLEAR Learning if more time is needed.
Does CLEAR provide CEUs?▼
CLEAR does not provide CEUs directly. Participants should contact clearlearning@clearhq.org for a course syllabus with outlined hours.
Is there a next level after NCIT Basic or Specialized?▼
There is no next NCIT level. Stand-alone Tier 3 modules are available — see clearhq.org.

Reference
NCIT Exam — Frequently Asked Questions
▼

Use these answers during Q&A sessions or when responding to participant emails about the exam.

What is the exam format?▼
The exam is pass/fail — no direct score is given to participants.
Can participants schedule for the same day?▼
No. Prometric does not allow same-day scheduling — the earliest test date is the next day.
When does the testing window open and close?▼
The testing window is 2 weeks, beginning the day of the Program Review Session. Advise participants not to schedule on the last day of the window.
Can a participant test in another state?▼
Yes, if their agency allows it.
How can participants check their system compatibility?▼
Direct them to https://rpcandidate.prometric.com/ — this tests their device only. They should also confirm their agency firewall/IT separately.
What ID is accepted?▼
Driver's license or passport is recommended. Large jewelry must be removed per Prometric security policy. No laptop on a docking station.
When will participants receive their digital badge?▼
Issued within a few business days after the testing window closes — no later than 3 weeks from the Review Session.
What is required for the NCIT Basic certificate?▼
Passing the exam plus 1 year of regulatory investigation/inspection experience. Records do not expire if the participant waits to apply.
How does the Specialized badge/certificate work?▼
Pass 1–2 sections → individual badges for those sections. Pass all 3 → one badge listing all modules. No expiration on NCIT Basic or Specialized certificate.
What is the retake policy for NCIT Basic?▼
If a participant fails, they may retake through Prometric after paying the $100 rescheduling fee. They do not need to retake the full course.
What is the retake policy for NCIT Specialized?▼
  • Fail 1–2 sections → retake internally with a CLEAR staff member ($50 for 1 section, $75 for 2).
  • Fail all 3 → retake through Prometric ($100 fee).
When does learning portal access close?▼
Access closes at the end of the testing window. Participants should download the eBook before then. Supplemental course recordings remain available until the course expires.
Will participants be tested on anything not in the eBook?▼
No. Participants will not be tested on anything not covered in the eBook/participant guide. Knowledge Assessments are locked and allowed only once — similar to exam questions. No practice tests are available. The Review Session is not recorded.
What if the name on the certificate is wrong?▼
The badge matches the name provided for the exam. Participants should contact CLEAR after receipt to make changes. Note: Prometric is instructed not to check middle names; if the middle name appears on the same line as the first name, the participant should email clearlearning@clearhq.org.
What if a module shows incomplete in the portal?▼
Contact CLEAR Learning — staff can mark it complete.
Can participants get an extension?▼
Reach out to CLEAR Learning if more time is needed.
Does CLEAR provide CEUs?▼
CLEAR does not provide CEUs directly. Participants should contact clearlearning@clearhq.org for a course syllabus with outlined hours.
Is there a next level after NCIT Basic or Specialized?▼
There is no next NCIT level. Stand-alone Tier 3 modules are available — see clearhq.org.

All Programs
Universal Session Facilitation Guidelines
▼

These guidelines apply to every session across every program. Review before each session.

1.1

Time Zones

All session times in this guide are in Eastern Time (ET). The program agenda sent to participants reflects the exact time zone in effect for each cohort. Always confirm the current time zone before each session and communicate it clearly to participants.

1.2

Instructors

Instructor names and bios are not listed in this guide, as instructors vary by cohort. Before each session:

  • Confirm the assigned instructor with the Program Coordinator.
  • Look up their bio in the Instructor Bios reference document (SharePoint → CLEAR Learning Facilitation).
  • Have the bio ready to read aloud during the introduction.
  • Fill in [Instructor Name] and [Instructor Bio] placeholders in the scripts below.
1.3

Before Every Session — Pre-Session Checklist

  • Log into Zoom 15 minutes early.
  • Admit the instructor from the waiting room and make them the Co-Host.
  • Confirm all polls are loaded in Zoom. If missing, create them before participants arrive.
  • For sessions with videos, have all video links open and tested in your browser.
  • Have the participant guide PDF ready to drop into chat the moment the instructor begins speaking.
  • Follow along with the slide deck throughout the session.
1.4

During Every Session — Ongoing Responsibilities

  • Monitor the chat continuously for raised hands, direct messages, and technical issues.
  • If the instructor asks whether anything is in the chat, respond promptly (or confirm it is empty).
  • When participants respond to instructor questions in chat: wait for a natural pause, read aloud, and confirm it has not already been mentioned.
  • Post time reminders in the chat as needed.
  • Camera: turn off after your opening remarks; turn back on for closing.
1.5

Standard Housekeeping Script

Script "Before I turn things over to the instructor, just a few housekeeping notes: If you haven't yet, we encourage you to turn your camera on, if you're able. If you need to turn it off at any time, that is completely fine. Please keep your line muted when you're not speaking to help minimize background noise. If you have any questions over today's content, please do not hesitate to ask. If you're not comfortable unmuting, you can send your questions or comments to me directly via chat. If you'd like to unmute, please use the Raise Hand feature under the Reactions button on Zoom. When sending a chat message, toggle the blue name to CLEAR LEARNING — not to the instructor, as they will not be monitoring the chat box. I'll also be placing the participant guide in the chat shortly — it's also available in the learning portal for your reference."
1.6

Instructor Introduction Script

Script "Now I'd like to introduce our instructor for today, [Instructor Name]. [Read bio from Instructor Bios reference document.] Please do not hesitate to ask any questions you might have — with that being said, I will turn it over to you, [Instructor Name]!"
1.7

Sharing the Participant Guide

During the first session of a program, as soon as the instructor begins speaking, paste the participant guide PDF into the Zoom chat.

1.8

Break Script & Chat Reminder

Break Script "We're going to pause here for a short break — about 10 minutes. Please stay on the session, and I'll put the return time in the chat for you. See you back at [return time, ET]!"
Post-Break Return Script "Welcome back, everyone! Hope you had a chance to stretch. If you could please turn your cameras back on, we are going to resume the session whenever you are ready, [Instructor Name]."

Legal Disclaimer Script

CLEAR-NCIT strongly recommends that all questions concerning the applicability of this material be resolved with your designated legal counsel before exercising any official authority."
1.14

Standard Closing Script Variations

General Closing "Thank you all for your participation in today's session. Please continue working on the self-study materials. Please do not hesitate to reach out if you have any questions."
With Evaluation Reminder "Thank you all for your participation in today's session. There will be an evaluation that appears in Zoom once I close the session. Please do not hesitate to reach out if you have any questions."
With Certificate Reminder "Thank you all for your participation in today's session. You will receive an email from the learning portal on how to download your certificate of completion."
